# Metarake

Metarake is a Rake extension to build multiple separate projects,
which are published outside the repository.

## Installation

Add this line to your application's Gemfile:

    gem 'metarake'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install metarake
